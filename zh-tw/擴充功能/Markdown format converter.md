Markdown 格式轉換工具能夠幫助您將其他的 Markdown 格式轉換成 Obsidian 格式。

目前您能夠轉換：

- Roam #tags 至 [[tags]]
- Roam `^^highlight^^` 至 `==highlight==`
- Bear `::highlight::` 至 `==highlight==`.
- Zettelkasten `[[UID]]` 連結至 full links `[[UID 檔名]]`
- Zettelkasten `[[UID]]` 連結至 pretty links `[[UID 檔名|檔名]]`，將顯示為 `檔名`
